## 📜 Description

Beginner-friendly personal portfolio website built with HTML, CSS & JavaScript.

This is my personal **portfolio website** showcasing who I am, my skills, projects, and services.
It is a multi-page website with a clean design, smooth animations, and a fully responsive layout.
Built entirely with **HTML, CSS, and JavaScript** — no frameworks, just pure web basics! 💪

---

## ✨ Features

- 🏠 **Home Page** – Introduction with animated skill progress bars
- 👩‍💻 **About Page** – Education timeline and personal bio
- 🚀 **Projects Page** – Showcase of GitHub projects with links
- 🛠 **Services Page** – Services offered (Web Dev, Data Science, AI/ML, Cloud)
- 📬 **Hire Me Page** – Direct links to GitHub and email
- 🎨 Smooth fade-in animations using JavaScript
- 📱 Responsive navigation across all pages

---

## 🛠 Tech Stack

- **Language:** HTML5, CSS3, JavaScript
- **Concepts Used:** Flexbox, CSS Animations, DOM Manipulation, Progress Bars, Multi-page Navigation

**Clone the repository**

git clone https://github.com/thepratikshamishra/portfolio.git
**Navigate to the project folder**

**Open in browser**
Open index.html in any web browser
